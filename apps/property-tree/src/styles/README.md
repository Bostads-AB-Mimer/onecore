# Styles

Global styles, CSS variables, and theme definitions.

## Structure

```
styles/
├── globals.css
├── variables.css
└── themes/
```

## Guidelines

- Global CSS resets and base styles
- CSS custom properties (variables)
- Theme definitions (light/dark mode)
- Component-specific styles should stay with their components

## Import Rules

This folder contains CSS only - no code import rules apply.
